Tailwind CSS v4 正式发布，这是一次重大的架构重写。新版本采用了全新的 Rust 编写的引擎 Oxide，带来了显著的性能提升。

## 核心变化

1. **性能飞跃**：全量构建速度提升 10 倍，增量构建提升 100 倍
2. **CSS-first 配置**：不再需要 `tailwind.config.js`，直接在 CSS 中使用 `@theme` 配置
3. **自动内容检测**：无需手动配置 `content` 路径
4. **原生级联层**：使用 CSS `@layer` 规范

对于正在使用 Tailwind CSS 的项目来说，升级到 v4 将带来更好的开发体验和更快的构建速度。
